Challenge

The main challenges of this project centered around environment setup and implementation details. ASan linking failed under MSYS2, VS Code could not locate the compiler, and paths containing Chinese characters and OneDrive caused image IO issues. Random jitter introduced brightness shifts, error diffusion could exceed bounds, and early contrast adjustments produced no visible difference. These were resolved by switching to Ubuntu, pointing to the correct g++, completing headers and clamping, and using zero-mean jitter, center-aligned sampling, and mean-centered brightness scaling.
