Challenge

The main challenges of this project centered around the environment and implementation: ASan linking failed under MSYS2, VS Code could not find the compiler, and paths with Chinese characters and OneDrive caused image reading errors. Random jitter produced brightness shifts, Floyd–Steinberg diffusion could exceed bounds, and early contrast changes had no visible effect. These issues were resolved by switching to Ubuntu, pointing to the correct g++, completing headers and clamping, using zero-mean jitter, center-aligned sampling, and scaling around mean brightness.
